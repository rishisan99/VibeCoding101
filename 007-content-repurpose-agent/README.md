# Content Repurpose Agent

Simple Streamlit app to convert one text into another format.

## Problem
One piece of content needs many manual rewrites.

## Solution
Pick input/output formats (blog, mail, post, tweet, script) and run one conversion.

## Setup
```bash
uv sync
```

## Run
```bash
uv run streamlit run app/main.py
```

## How to Use
1. Enter OpenAI API key in sidebar.
2. Paste source text in Input box.
3. Choose input and output formats.
4. Click `Run`.
5. Use `Copy` button to copy result.
